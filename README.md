Hello 8000 Chrome 扩展插件
=====

### 主要功能

和那些 FK8000 不一样的地方在于,首先我们有礼貌,其次我们能搞定网站白名单但JS/CSS所在CDN被拦截导致无法使用的高阶问题.

1. 自动点[访问30分钟]
2. 修正 `Default.aspx?__tencentip=&__tencentid=&__tencentrawurl=` 问题(也就是点完后没有跳回正确的页面)
3. 登陆页面: 记住用户名,记录登陆类型,聚焦密码框
4. 我什么都不知道
